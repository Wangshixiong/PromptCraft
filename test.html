PromptCraft 插件测试页面

测试说明

这个页面用于测试 PromptCraft 插件的功能。请按照以下步骤进行测试：

  1. 确保已经在 Chrome 中加载了 PromptCraft 插件
  2. 使用快捷键 Ctrl+B (Mac: Cmd+B) 打开侧边栏
  3. 点击"使用 GitHub 登录"按钮测试登录功能
  4. 选中下面的测试文本，右键选择"添加到 PromptCraft"

测试文本

AI 提示词示例 1

请扮演一个专业的产品经理，帮我分析以下产品需求的可行性和优先级。需要考虑技术实现难度、用户价值、商业价值和资源投入等因素。

AI 提示词示例 2

作为一名经验丰富的前端开发工程师，请帮我优化这段 JavaScript 代码的性能，并解释优化的原理和预期效果。

AI 提示词示例 3

请以创意写作专家的身份，为我的科幻小说构思一个引人入胜的开头，背景设定在2050年的智能城市中。

功能检查清单

  • □ 侧边栏能正常打开和关闭
  • □ GitHub 登录功能正常工作（无 302 错误）
  • □ 右键菜单"添加到 PromptCraft"可见
  • □ 选中文本后右键添加功能正常
  • □ 提示词列表能正常显示
  • □ 搜索功能正常工作
  • □ 主题切换功能正常

故障排查

如果遇到问题，请：

  1. 打开浏览器开发者工具 (F12)
  2. 查看 Console 标签页中的错误信息
  3. 检查 Network 标签页中的网络请求状态
  4. 确认 Supabase 配置是否正确
